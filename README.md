# 🚀 Forecasting the Unpredictable: Nike’s Inventory Intelligence System

---

## 📌 Overview

Managing inventory in a dynamic retail environment is a complex challenge. Demand fluctuates due to seasonality, promotions, and customer behavior, making it difficult for companies like Nike to maintain the right balance between supply and demand.

This project presents an **AI-powered, interactive Inventory Intelligence System** that not only predicts demand but also helps businesses make **data-driven decisions** through insights, alerts, and recommendations.

---

## 🎯 Problem Statement

Retail companies face multiple challenges:

* 📉 Unpredictable customer demand
* 📦 Overstocking and stockouts
* 🏬 Poor inventory alignment across stores
* ⚡ Lack of real-time decision-making support
* 😟 Reduced customer satisfaction

---

## ⚠️ Gap Analysis (Critical Insight)

| Current Approach        | Limitations          | Gap                        |
| ----------------------- | -------------------- | -------------------------- |
| Traditional forecasting | Inaccurate & delayed | No real-time intelligence  |
| Static dashboards       | Only visualization   | No decision-making support |
| Manual planning         | Time-consuming       | No automation or insights  |

👉 **Key Gap:**
Existing systems show *what is happening*, but not *what to do next*.

---

## 💡 Our Solution

We developed an **AI-powered Decision Intelligence System** that transforms raw data into actionable business insights.

### Key Capabilities:

* 📈 Predict future demand using Machine Learning
* 🧠 Generate AI-driven insights dynamically
* ⚠️ Detect risks with smart alerts
* 🎯 Recommend actionable inventory decisions
* 🔄 Allow real-time user interaction and simulation

---

## ⚙️ Core Features

### 🔹 1. Interactive Dashboard

* Store & Department selection
* Date range filtering
* Holiday / Promotion toggle
* Real-time updates

---

### 🔹 2. Demand Forecasting (ML-Based)

* Uses Machine Learning model (Random Forest)
* Predicts future demand trends
* Handles seasonality and historical patterns

---

### 🔹 3. What-if Simulation (Advanced Feature)

Users can simulate:

* Demand increase/decrease
* Promotions
* Inventory adjustments
* Seasonal scenarios

👉 Instantly updates:

* Forecast
* Risk level
* Recommendations

---

### 🔹 4. Smart Alert System

* 🔴 High demand alerts
* 🟡 Overstock risk
* 🟢 Low stock warnings

👉 Includes:

* Clickable details
* Reason behind alerts
* Suggested actions

---

### 🔹 5. AI Insights Engine

Auto-generated insights like:

* Demand spikes in specific stores
* Seasonal trends
* Performance comparisons

---

### 🔹 6. Recommendation Engine

Provides actionable suggestions:

* Increase inventory
* Balance stock across stores
* Reduce overstock

---

### 🔹 7. Decision Summary Panel

Clear output for business decisions:

* Suggested inventory change (%)
* Risk level
* Expected impact

---

### 🔹 8. Explainable AI

* Shows feature importance
* Explains prediction logic
* Improves trust & transparency

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Machine Learning:** Scikit-learn (Random Forest)
* **Data Processing:** Pandas, NumPy
* **Visualization:** Plotly

---

## 🔄 How It Works

1. Load historical sales dataset
2. Perform preprocessing & feature engineering
3. Train ML model for demand prediction
4. Apply user filters and simulations
5. Generate:

   * Forecasts
   * Insights
   * Alerts
   * Recommendations
6. Display everything in an interactive dashboard
